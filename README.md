# Web-tool-to-generate-PBR-ORMA-ARM-inZOI
 I've created a free web tool to generate PBR (ORMA/ARM) textures optimized for inzoi and Unreal Engine 5!  You can instantly create Occlusion, Roughness, and Metallic maps from a base image, with automatic presets (fabric, metal, plastic) and real-time preview.
